

## EXP 5: Comparative Analysis of Different Types of Prompting Patterns and Explain with Various Test Scenarios
# Aim

To test and compare how different prompt pattern models respond to naïve prompts (broad or unstructured) versus basic prompts (clear and refined) across multiple scenarios and analyze the quality, accuracy, and depth of the generated responses.

# AI Tools Required

ChatGPT

Internet Browser

Computer System

# Explanation
# 1. Define the Two Prompt Types
# Naïve Prompt

A naïve prompt is simple, vague, or unstructured. It gives minimal instructions and leaves the AI to interpret the request.

Example:
"Write about artificial intelligence."

# Basic Prompt

A basic prompt is clear, detailed, and structured. It provides specific instructions, context, or format to guide the AI.

Example:
"Explain artificial intelligence in 150 words including its definition, applications, and advantages."

# 2. Test Scenarios

Different scenarios were used to evaluate how ChatGPT responds to naïve and basic prompts.

Scenarios include:

Creative Story Generation

Answering a Factual Question

Summarizing a Concept

Providing Advice or Recommendations

Each scenario uses both naïve and basic prompts to perform the same task.

# 3. Experimental Procedure

Select a scenario.

Provide a naïve prompt to ChatGPT.

Record the generated response.

Provide the basic prompt for the same scenario.

Record the generated response.

Compare results based on quality, accuracy, and depth.

# 4. Test Scenarios with Prompts
Scenario 1: Creative Story Generation
Naïve Prompt

Write a story about a lion.

ChatGPT Output

A lion lived in a forest and ruled over all animals. One day he helped a small rabbit and learned that kindness is important. The animals respected him and lived happily.

Basic Prompt

Write a short creative story (150 words) about a brave lion that protects other animals in the jungle. Include a moral at the end.

ChatGPT Output

In a dense jungle lived a brave lion named Leo. Unlike other lions, Leo believed that strength should be used to protect others. One day hunters entered the jungle and scared the animals. Leo courageously roared and chased them away. The animals felt safe again and thanked him for his bravery. From that day on, Leo became the guardian of the jungle.
Moral: True strength lies in protecting others.

Scenario 2: Factual Question
Naïve Prompt

What is machine learning?

ChatGPT Output

Machine learning is a branch of artificial intelligence where computers learn from data and improve their performance without being explicitly programmed.

Basic Prompt

Explain machine learning in simple terms including its definition, types, and real-life applications in about 120 words.

ChatGPT Output

Machine learning is a field of artificial intelligence that allows computers to learn from data and improve automatically without being explicitly programmed. It works by analyzing patterns in data and making predictions or decisions. There are three main types of machine learning: supervised learning, unsupervised learning, and reinforcement learning. Supervised learning uses labeled data, while unsupervised learning finds patterns in unlabeled data. Reinforcement learning improves through trial and error. Machine learning is widely used in real-life applications such as recommendation systems (Netflix, Amazon), speech recognition, medical diagnosis, and fraud detection.

Scenario 3: Summarizing a Concept
Naïve Prompt

Explain cloud computing.

ChatGPT Output

Cloud computing is a technology that allows users to store and access data and applications over the internet instead of using local computers.

Basic Prompt

Summarize cloud computing in 5 points including definition, benefits, examples, and common services.

ChatGPT Output

Definition: Cloud computing is the delivery of computing services such as storage, servers, and software over the internet.

Benefits: It reduces hardware cost and provides scalability.

Accessibility: Users can access data from anywhere with an internet connection.

Examples: Google Drive, AWS, Microsoft Azure.

Services: Includes Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS).

Scenario 4: Advice / Recommendation
Naïve Prompt

How to stay healthy?

ChatGPT Output

To stay healthy, eat nutritious food, exercise regularly, drink water, and get enough sleep.

Basic Prompt

Give 5 practical tips for college students to maintain a healthy lifestyle including diet, exercise, sleep, and stress management.

ChatGPT Output

Balanced Diet: Eat fruits, vegetables, and protein-rich foods while avoiding junk food.

Regular Exercise: Engage in physical activity like walking, yoga, or sports for at least 30 minutes daily.

Proper Sleep: Maintain a sleep schedule and get 7–8 hours of sleep each night.

Hydration: Drink enough water to stay energized and focused.

Stress Management: Practice meditation, hobbies, or time management to reduce academic stress.

# 7. Analysis

The experiment shows that prompt clarity significantly improves AI-generated responses.

Naïve prompts produce shorter and less detailed outputs because they lack clear instructions.

Basic prompts guide the AI by providing structure, context, and formatting requirements.

As a result, basic prompts generate responses that are more informative, structured, and comprehensive.

However, naïve prompts may still work well for simple factual questions, where detailed instructions are not necessary.

# 7. Findings

Structured prompts produce higher quality responses.

Basic prompts improve accuracy and completeness.

Clear instructions increase depth and organization of responses.

Naïve prompts are suitable for simple queries but not for complex tasks.

# Result

The comparative analysis of naïve and basic prompting patterns was successfully performed using ChatGPT. The experiment demonstrates that well-structured prompts significantly improve the quality, accuracy, and depth of AI-generated responses.
